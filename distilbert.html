Fine-tuning DistilBERT
漏洞分類模型.

利用預訓練語言模型 DistilBERT 微調，針對資安漏洞描述文本建立多分類模型，可辨識 CWE 漏洞類型，顯著提升威脅偵測精準度。

01

任務定義

此為多分類的句子分類任務（Sentence Classification）， 輸入資安漏洞描述文本，輸出對應的 CWE 類別標籤。

→ 查看完整 Jupyter Notebook 程式碼

02

資料集

  • 來源：Kaggle — NVD 作業系統漏洞資料，共 168,000 筆
  • 採樣 10,000 筆訓練 + 1,000 筆測試
  • 資料格式：CVE / CWE / CPE 結構化標準
  • 分類標籤來自 CWE (Common Weakness Enumeration)
03

資料預處理

刪除缺失值、保留關鍵欄位後，用 DistilBERT tokenizer 將文本轉為模型輸入格式，再切分固定長度區塊。

tokenize_function(example)
group_texts(examples)   # 切分固定長度區塊

從 CWE 字串擷取數字部分作為類別 index，完成標籤處理。

04

模型選擇

選用 Hugging Face 的 DistilBERTForSequenceClassification：

  • 比 BERT 輕量，訓練速度快且表現穩定
  • 保留 BERT 的語意理解與 contextual embedding 優勢
  • 適合中型語料與多分類任務
  • 使用 full-mask 預訓練策略進行 fine-tuning
05

後續展望

  • 搭配自動爬取 CVE 描述，實現即時漏洞風險分類
  • 增加訓練資料量以進一步提升準確率
  • 嘗試完整 BERT 或 RoBERTa 以比較效能
